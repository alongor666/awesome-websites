## ✨ 特性

- 🗂️ **分类清晰** - 按用途分类，便于查找
- 🚀 **精选优质** - 只收录高质量、实用性强的网站
- 🔄 **定期更新** - 持续维护，添加新的优质资源
- 📝 **详细说明** - 每个网站都有简要说明
- 🌐 **中文友好** - 优先收录对中文用户友好的网站
